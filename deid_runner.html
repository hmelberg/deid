SDC verbs → microdata.no script

Write disclosure-control steps in a small call syntax — e.g. round(inntekt, to=1000) — and deid translates them to a valid microdata.no script (shown in Generated microdata).

Because microdata can't run in the browser, the Effect on sample data panel simulates each step on a tiny example table so you can see the before → after (changed cells highlighted, dropped rows struck out). Random verbs and a few others aren't simulated — they still translate.

The sidebar lists every verb: click Try to load an example, or Docs to read what it does.

Tip: a list as the first argument applies a verb to several columns — round([inntekt, formue], to=1000). Any verb may carry an apply_ prefix.

Type SDC verbs (one per line). Set the unit_id for per-unit random verbs. Output updates live.
